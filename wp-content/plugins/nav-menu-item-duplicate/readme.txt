A simple plugin that adds a duplicate button to each items on edit menu screen.

== Description ==
This Plugin will save your time by adding a duplicate button on each menu item while editing a menu. A real time saver. Currently it doesn\'t support customize screen, but I\'ll add it soon. If you find any bugs, just head over to support forum, I\'ll fix that asap.

Please rate me a 5 star if you find this plugin helpful :)

== Installation ==
Method 1:

1. Go to Plugins > Add New
2. Search for \"Nav Menu Item Duplicator\"
3. Install & Activate

Method 2:

1. Download the plugin .zip file from wordpress.org repository
2. Go to Plugins > Add New and click upload plugin
3. Browse the plugin .zip file and install
4. Click on Activate Plugin
